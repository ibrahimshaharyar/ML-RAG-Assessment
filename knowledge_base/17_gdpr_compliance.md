# GDPR Compliance

We are committed to protecting your personal data and complying with the General Data Protection Regulation (GDPR).

## Data We Collect

| Category | Examples | Purpose |
|----------|---------|---------|
| Account data | Name, email, password hash | Authentication |
| Usage data | API calls, query logs | Service delivery |
| Billing data | Payment method, invoices | Billing |
| Device data | IP address, browser | Security |

## Legal Basis for Processing

We process your data under these lawful bases:
- **Contract**: To provide the services you signed up for
- **Legitimate interest**: Security monitoring, fraud prevention
- **Consent**: Marketing emails (opt-in only)
- **Legal obligation**: Financial records retention

## Your Rights Under GDPR

| Right | How to Exercise |
|-------|----------------|
| Access | Email privacy@example.com |
| Rectification | Update in Settings → Account |
| Erasure | Account deletion (Settings → Account → Delete) |
| Portability | Data export (Settings → Data & Privacy) |
| Object | Email privacy@example.com |
| Restrict processing | Email privacy@example.com |

We respond to all GDPR requests within **30 days**.

## Data Retention

| Data Type | Retention Period |
|-----------|-----------------|
| Account data | Until account deletion |
| Query logs | 90 days (Pro) / 30 days (Starter) |
| Billing records | 7 years (legal requirement) |
| Security logs | 1 year |

## Data Processors

We use these third-party processors, all GDPR-compliant:
- **Stripe** — payment processing
- **AWS** — infrastructure & storage
- **SendGrid** — transactional emails
