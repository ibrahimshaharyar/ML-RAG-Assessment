# Integrations Overview

Connect our platform with the tools your team already uses.

## Available Integrations

### Productivity
| Integration | What It Does |
|------------|-------------|
| **Slack** | Get query answers and alerts in Slack |
| **Microsoft Teams** | Query the knowledge base directly from Teams |
| **Notion** | Sync knowledge base documents from Notion |

### Observability
| Integration | What It Does |
|------------|-------------|
| **Datadog** | Send usage metrics and latency data |
| **PagerDuty** | Alert on service degradation |
| **New Relic** | APM and infrastructure monitoring |

## Setting Up an Integration

1. Go to **Settings → Integrations**
2. Find the integration and click **"Connect"**
3. Follow the OAuth flow or enter credentials
4. Configure which events/channels to use
5. Test the integration with a test event

## Managing Connected Integrations

View all active integrations in **Settings → Integrations → Connected**.
- Click any integration to view its settings or disconnect it
- Disconnecting is immediate and revokes all access tokens
