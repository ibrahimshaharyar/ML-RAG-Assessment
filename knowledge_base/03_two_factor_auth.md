# Two-Factor Authentication (2FA)

Two-Factor Authentication adds an extra layer of security to your account. Even if your password is compromised, your account remains protected.

## Enabling 2FA

1. Log in to your account
2. Navigate to **Settings → Security**
3. Under **Two-Factor Authentication**, click **"Enable 2FA"**
4. Choose your preferred method:
   - **Authenticator App** (recommended) — Google Authenticator, Authy
   - **SMS** — receive a code via text message
5. If using an app, scan the QR code shown on screen
6. Enter the 6-digit code generated by the app to confirm
7. Save your **backup codes** in a safe place

## Logging In With 2FA

When 2FA is enabled:
1. Enter your email and password as usual
2. You will be prompted for your 2FA code
3. Open your authenticator app and enter the current 6-digit code
4. Click **"Verify"**

## Backup Codes

Backup codes can be used if you lose access to your authenticator app. Each code can only be used **once**.

- To view or regenerate codes: **Settings → Security → Backup Codes**
- Store them offline in a secure location

## Disabling 2FA

1. Go to **Settings → Security**
2. Click **"Disable 2FA"**
3. Confirm with your password and a current 2FA code

> ⚠️ **Note:** Disabling 2FA reduces your account security. We do not recommend it.

## Troubleshooting

- **Code not working**: Ensure your device's clock is synced correctly (2FA codes are time-based)
- **Lost your device**: Use a backup code to regain access, then re-enroll 2FA
- **Can't access backup codes**: Contact support with account verification details
